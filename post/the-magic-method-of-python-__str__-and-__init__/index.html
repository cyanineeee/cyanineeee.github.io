The magic method of python : __str__ and __init__

2022-10-02 1 min read # python

在一个类中，__init__方法会在类的初始化时自动调用
__str__会在使用print(类)的时候调用

example:

class Cat():
    def __init__(self,name):
        self.name=name
    def __str__(self):
        return self.name
    def hello(self):
        print('hello,{}'.format(self.name))

cat = Cat('mimi')
print(cat)
cat.hello()
print(cat.name)

运行结果：
mimi
hello,mimi
mimi
